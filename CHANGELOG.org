#+AUTHOR: Paul H. McClelland

All notable changes to ~outline-stars~ are documented in this file.

* 0.3.0 (2026-04-03)

- Pre-seeded ~outline-heading-alist~ for fast level lookup: heading level is now resolved via alist lookup instead of regexp matching on every call, improving navigation performance in large files
- New customization ~outline-stars-level-1-overline~ (default ~nil~): when non-nil, top-level headings are fontified with an overline, creating a visual section break
- New command ~outline-stars-number-headings~: inserts or updates hierarchical section numbers (1, 1.1, 1.1.2) between the stars and heading text; idempotent (strips existing numbers before re-numbering)
- New command ~outline-stars-strip-numbers~: removes section numbers from all headings
- ~outline-stars-setup~ now sets ~outline-minor-mode-cycle~ buffer-locally, enabling TAB cycling on headings without requiring manual configuration

* 0.2.0 (2026-03-31)

- New customization ~outline-stars-section-comments~ (default ~t~): uses section-level comment prefixes (=;;; *=, =### *=) for modes with single-character comment starters; multi-character starters like =//= are unaffected, avoiding Doxygen conflicts; set to nil for classic ~outshine~ behavior (=;; *=, =## *=)
- New command ~outline-stars-cycle-buffer~: cycle global visibility through show all, headings only, and top-level, from any point in the buffer
- New command ~outline-stars-sort-siblings~: alphabetically sort sibling headings under the current parent, carrying their subtrees (prefix argument for reverse order)
- New customization ~outline-stars-promote-subtree-p~: when non-nil (the default), ~outline-stars-promote~ and ~outline-stars-demote~ operate on the entire subtree; when nil, only the current heading is changed
- ~outline-stars-promote~ and ~outline-stars-demote~ now consult ~outline-stars-promote-subtree-p~; explicit ~outline-stars-promote-subtree~ and ~outline-stars-demote-subtree~ remain available for direct use

* 0.1.0 (2026-03-28)

Initial release.

- Global minor mode ~outline-stars-mode~ for automatic activation
- Comment-prefix calculation via ~comment-start~ and ~comment-add~
- Per-level fontification with ~outline-stars-level-N~ faces
- Imenu integration for heading navigation
- Structure editing: ~outline-stars-insert-heading~, ~outline-stars-promote-subtree~, ~outline-stars-demote-subtree~, ~outline-stars-narrow-to-subtree~
- Nullifies ~outline-search-function~ (Emacs 29+) for correct regexp handling
- Runs on ~after-change-major-mode-hook~ to override mode-specific defaults
