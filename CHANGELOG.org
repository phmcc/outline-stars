#+TITLE: Changelog
#+AUTHOR: Paul H. McClelland
#+OPTIONS: toc:0 num:nil

All notable changes to ~outline-stars~ are documented in this file.

* 0.4.1 (2026-04-18)

- ~outline-stars-insert-heading~ now normalizes spacing around the new heading to match the document convention: detects blank-line spacing from the nearest heading, then applies that spacing both above and below the new heading regardless of how much whitespace currently exists at point
- Promote/demote commands now leave point at end of the current heading line after renumbering
- Removed unused parameter from ~outline-stars--next-number~
- ~outline-stars--next-number~ returns nil gracefully when no heading exists
- ~outline-stars--maybe-teardown~ checks ~outline-stars--active~ instead of ~outline-stars--font-lock-keywords~
- Trimmed docstrings throughout

* 0.4.0 (2026-04-11)

- ~outline-stars-level-1-overline~ replaced by ~outline-stars-overline~: accepts ~nil~ (off), ~level-1~ (top-level only), or ~t~ (all levels); each level's overline colored from its face foreground
- ~outline-stars-auto-number~ (default ~t~): auto-inserts section numbers on heading insertion; promote/demote/sort renumber automatically
- ~outline-stars-insert-heading~ detects blank line spacing contextually (mirrors heading spacing when on a heading, reuses blank lines when in body text)
- New command ~outline-stars-number-subtree~: numbers children within the current parent's subtree
- New command ~outline-stars-number-region~: numbers headings in the active region, inheriting from the last numbered heading before the region
- Numbering refactored into ~outline-stars--number-in-region~ / ~outline-stars--strip-numbers-in-region~ with ~base-level~ and ~start-offset~ parameters
- ~outline-stars--next-number~ reads the current heading's number and increments the last component (works at any depth)
- Promote/demote renumber the parent subtree (not the whole buffer)
- Parent-subtree renumber inherits the parent's full number as a prefix and pads implied intermediate levels

* 0.3.0 (2026-04-04)

- Pre-seeded ~outline-heading-alist~ for fast level lookup via alist instead of regexp matching on every call
- New customization ~outline-stars-level-1-overline~ (default ~nil~): overline on top-level headings for visual section breaks
- New customization ~outline-stars-default-state~ (default ~nil~): initial buffer visibility—~folded~ (top-level only), ~content~ (all headings, body hidden), ~show-all~, or ~nil~ (no action); safe for use as a file-local variable
- New customization ~outline-stars-default-state-alist~ (default ~nil~): per-mode overrides for default visibility, checked before the global default
- New customization ~outline-stars-number-start~ (default ~1~): starting index for section numbering; set to 0 for zero-based numbering
- New command ~outline-stars-number-headings~: insert or update hierarchical section numbers; idempotent
- New command ~outline-stars-strip-numbers~: remove section numbers from all headings
- Default state application deferred to ~hack-local-variables-hook~ so file-local overrides take effect
- ~outline-stars-setup~ now sets ~outline-minor-mode-cycle~ buffer-locally, enabling TAB cycling without manual configuration

* 0.2.0 (2026-03-31)

- New customization ~outline-stars-section-comments~ (default ~t~): uses section-level comment prefixes (=;;; *=, =### *=) for modes with single-character comment starters; multi-character starters like =//= are unaffected, avoiding Doxygen conflicts; set to nil for classic ~outshine~ behavior (=;; *=, =## *=)
- New command ~outline-stars-cycle-buffer~: cycle global visibility through folded, content, and show all from any point in the buffer
- New command ~outline-stars-sort-siblings~: alphabetically sort sibling headings under the current parent, carrying their subtrees (prefix argument for reverse order)
- New customization ~outline-stars-promote-subtree-p~: when non-nil (the default), ~outline-stars-promote~ and ~outline-stars-demote~ operate on the entire subtree; when nil, only the current heading is changed
- ~outline-stars-promote~ and ~outline-stars-demote~ now consult ~outline-stars-promote-subtree-p~; explicit ~outline-stars-promote-subtree~ and ~outline-stars-demote-subtree~ remain available for direct use

* 0.1.0 (2026-03-28)

Initial release.

- Global minor mode ~outline-stars-mode~ for automatic activation
- Comment-prefix calculation via ~comment-start~ and ~comment-add~
- Per-level fontification with ~outline-stars-level-N~ faces
- Imenu integration for heading navigation
- Structure editing: ~outline-stars-insert-heading~, ~outline-stars-promote-subtree~, ~outline-stars-demote-subtree~, ~outline-stars-narrow-to-subtree~
- Nullifies ~outline-search-function~ (Emacs 29+) for correct regexp handling
- Runs on ~after-change-major-mode-hook~ to override mode-specific defaults
